# Demon Jewellery — Google Sites + Apps Script + GitHub

Tienda web con Google Apps Script y Google Sheets, preparada para incrustarse en Google Sites.

## Uso
1. Copia los archivos a un proyecto de Apps Script o sincroniza con clasp.
2. Ejecuta `setup()` una vez.
3. Configura correo y WhatsApp con `configureStore({adminEmail:'correo@dominio.com', whatsapp:'521XXXXXXXXXX'})`.
4. Implementa como aplicación web y pega la URL `/exec` en Google Sites.
